YASMINE WARE PORTFOLIO WEBSITE

This portfolio site is ready to host on GitHub Pages.

STEPS TO PUBLISH:
1. Go to GitHub and create a new public repository (e.g. yasmine-ware-portfolio).
2. Upload the contents of this folder (index.html, styles/, assets/, readme.txt).
3. Commit your changes.
4. Go to Settings → Pages → Branch: main → Save.
5. Your website will appear at: https://yourusername.github.io/yasmine-ware-portfolio

TO REPLACE FILES:
- Profile Photo: Replace /assets/profile.jpg (keep same filename and aspect ratio).
- Resume: Replace /assets/Yasmine_Ware_Resume.pdf.
- The "View My Resume" button will automatically open the new file.

TO CHANGE THE BUTTON LINK:
- Open index.html.
- Find the line: <a class="resume-btn" href="assets/Yasmine_Ware_Resume.pdf" target="_blank">.
- Replace the href with a new link (for example, a Google Drive or Dropbox URL).

Edit text directly in index.html and color styles in /styles/style.css.
